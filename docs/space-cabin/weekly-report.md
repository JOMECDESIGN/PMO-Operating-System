# RAG 周报 — 二代太空体验舱

> 宪法 weekly-report 的本项目落点。整体健康 = 最差分量。坏消息早、透明、带解法。数据,不靠感觉。

## 周报 6 段式
1. **整体健康**(RAG 一句话;坏消息带解法)。
2. **里程碑**(8 阶段 + 甲方确认点,带健康与判断)。
3. **关键路径**(哪些任务、状态、缓冲消耗天数)。
4. **风险告警**(触发/监控风险 + 动作)。
5. **本周做 / 下周计划**(对计划)。
6. **需决策**(谁、决什么、何时前)。

## 整体健康规则
**整体 = 最差分量**。任一里程碑红 / 任一高风险触发 / 关键路径缓冲耗尽 → 整体红。诚实的黄/红胜过装饰性的绿。
> 本项目特殊性:出厂验收前**结构性零浮动**,故整体健康**基线即 AMBER**,不存在"全绿"状态,除非 11/9 运行就绪后。
